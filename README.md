# CODIFICACIÓN HUFFMAN EN MATLAB

# Objetivos
1. Aprender cómo codificar y decodificar un alfabeto de mensajes usando los comandos de Matlab.
2. Calcular la entropía, longitud promedio del código, la cantidad de información de cada mensaje, la eficiencia y la redundandia de un código.
3. Verificar el primer teorema de Shannon.
